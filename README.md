# 📌 감정 기반 관계도 앱 (Emotion Graph App)

사용자의 감정을 시각적으로 표현하고 관리할 수 있는 감정 기반 관계도 앱입니다.  
React 기반으로 구축되었으며, React Query와 Zustand를 통해 상태 및 서버 상태를 효율적으로 관리합니다.
